Minhas Notas - App de Arquitetura Local
Um aplicativo web mobile-first de notas adesivas, construído com foco em extrema leveza, estabilidade e persistência de dados diretamente no dispositivo.
🚀 Propósito do Projeto
Este projeto foi desenvolvido para operar sem dependências complexas ou frameworks pesados que podem causar instabilidade. Adotamos uma Arquitetura Vanilla (arquivo único HTML/JS) para garantir um deploy imediato e seguro via GitHub Pages, ideal para o fluxo de trabalho direto pelo smartphone.
✨ Funcionalidades
Persistência Offline (localStorage): As notas são salvas automaticamente na memória do navegador do telemóvel. O conteúdo permanece lá mesmo após fechar a aba ou reiniciar o aparelho.
Interface Mobile-First: Design em grid responsivo que se ajusta com precisão a qualquer tela de smartphone.
Light Mode de Alto Contraste: Interface limpa e clara, priorizando a legibilidade absoluta e o realismo visual.
Gestão Ágil: Botão flutuante (FAB) para criação instantânea e módulos independentes para cada nota.
🏗️ Arquitetura e Tecnologias
Front-end: HTML5, CSS3 e JavaScript (Vanilla JS) integrados.
Sem Build: O código não precisa de compilação ou tradução; o navegador lê o arquivo original instantaneamente.
Hospedagem: Otimizado para GitHub Pages e ambientes estáticos.
Estrutura Modular: Cada nota é tratada como um componente isolado, garantindo que um erro em uma nota nunca trave o aplicativo inteiro.
🛠️ Como Utilizar
Basta carregar o arquivo index.html no seu repositório do GitHub.
Ative o GitHub Pages nas configurações do repositório.
Acesse o link gerado e comece a organizar suas ideias.
